
عنوان پروژه:
لندینگ پیج نرم‌افزار مدیریت پخش گل زعفران برای مدیران توزیع

هدف لندینگ پیج:
ترغیب مدیران پخش گل زعفران به ثبت شماره موبایل و درخواست مشاوره، با تمرکز بر افزایش فروش، کاهش ضایعات و شفافیت عملیات.


---

1. هدر (Header)

لوگو نرم‌افزار در بالا

عنوان جذاب: «مدیریت هوشمند پخش گل زعفران و افزایش بازدهی فروش»

زیرعنوان: «شفافیت کامل در سفارش، تحویل و تسویه با یک پلتفرم واحد»

CTA اصلی: «درخواست مشاوره» (لینک به فرم شماره موبایل)



---

2. بخش معرفی محصول (Hero Section)

تصویر/ویدئو: مدیر پخش در حال مدیریت سفارشات و پاک‌کنندگان در حال کار

مزایای کلیدی:

1. کنترل دقیق سفارش و تحویل گل


2. کاهش ضایعات و افزایش سود


3. شفافیت کامل در تسویه حساب‌ها


4. گزارش‌ها و تحلیل‌های مدیریتی برای تصمیم‌گیری سریع

---

7. ویژگی‌های فنی

HTML و CSS خام، آماده انتشار

فرم داده ها را به این صورت بفرستد:
<form id="consultForm">
  <input type="tel" name="phone" required placeholder="09xxxxxxxxx">
  <button type="submit">ارسال</button>
</form>

<script>
document.getElementById("consultForm").addEventListener("submit", async (e) => {
  e.preventDefault();
  const data = { phone: e.target.phone.value };
  await fetch("YOUR_SCRIPT_URL", {
    method: "POST",
    body: JSON.stringify(data),
    headers: { "Content-Type": "application/json" }
  });
  alert("درخواست ثبت شد ✅");
  e.target.reset();
});
</script>
